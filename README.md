### LearningCompose

Aplicativo desenvolvido com foco no aprendizado de Jetpack Compose, utilizando componentes como Row, Column, Surface e animações simples para criar cartões expansíveis.
O projeto demonstra como manipular estado, aplicar temas modernos e estruturar listas usando uma abordagem totalmente declarativa, facilitando o entendimento de como o Compose funciona no dia a dia do Android.

### Funcionalidades 
* Lista de mensagens usando LazyColumn
* Cartões expansíveis com animação
* Manipulação de estado com remember
* Animação de cor
* Layouts com Row, Column, Spacer e Surface

### Tecnologias utilizadas
* Kotlin
* Jetpack Compose
* Material 3
* Uso de componentes de estado e animação
* Android Studio
